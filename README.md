
# Solar System 

### Visão Geral
O Solar System Project é uma demonstração interativa do sistema solar e das missões espaciais já realizadas. 
O projeto foi desenvolvido com o objetivo de proporcionar uma experiência educativa e informativa sobre nosso sistema planetário e as explorações espaciais.

### Funcionalidades

- Explore o sistema solar de forma interativa, navegando entre os planetas e suas luas em um modelo tridimensional.


- Obtenha informações detalhadas sobre cada planeta, como suas características, atmosfera, satélites naturais e muito mais.


- Conheça as principais missões espaciais realizadas ao longo da história, com detalhes sobre seus objetivos e descobertas.


- Visualize imagens reais dos planetas, luas e das missões espaciais.
  
### Como Utilizar

1- Acesse o projeto através do link ou execute-o localmente após o download dos arquivos.

2- A interface do projeto apresentará uma visão geral do sistema solar, com o Sol e os planetas principais.

3- Utilize os controles interativos para navegar pelo sistema solar e selecionar planetas individuais para obter mais informações.

4- Ao selecionar um planeta, você verá detalhes sobre suas características, como tamanho, massa, distância do Sol, entre outros.

5- Explore a seção de missões espaciais para conhecer os marcos históricos das explorações espaciais humanas.

6- Divirta-se aprendendo e explorando o sistema solar e as missões espaciais realizadas!

### Personalização
O Solar System Project foi desenvolvido como uma demonstração educativa, mas você pode personalizá-lo para adicionar mais planetas, informações ou recursos adicionais, como vídeos, animações ou simulações de órbitas.

### Ferramentas utilizadas:

- React, CSS3, HTML5, ES6.
